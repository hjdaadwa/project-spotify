 <h3>Для проверки:</h3>
 <p>1. Установите зависимости</p>
 
```Bash
npm install
```
<p>2. Запустить прокси сервер для авторизации и хранения ключей в одном терминале. Запустить приложение в другом.</p>

Запусить локальный сервер
```Bash
node server
```


Запусить приложение
```Bash
npm start
```


<li>3. Открыть</li>
<b>http://localhost:3000/<b>
